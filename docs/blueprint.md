# **App Name**: Walert

## Core Features:

- Realtime Notification Display: Display location and timestamp from the Firebase Realtime Database as notifications appear.
- Data Fetching: Fetch location and timestamp data from Firebase Realtime Database.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Soft Purple (#9575CD) to add a touch of elegance without overwhelming the minimalist aesthetic.
- Font: 'Inter', a sans-serif font, for both body text and headlines due to its modern and clean appearance.
- Single-page layout with a focus on displaying notifications in a chronological order.
- Use simple, geometric icons to represent wallet opening and location events.
- Subtle fade-in animations as new notifications appear to provide smooth UI transitions.
